# TJ_MovieBrowser
Movie BROWSER

# Language : Swift 4
# Compatibility : iPhone's
# Design Pattern : MVVM(Model View ViewMode)
# Pagination
# API Call through NSURLSession Task
# Webservice handle by Webservice Handler 
# Functionality : 
  - Movie List
  - Sort Movie by Sort can be most popular and Higest rated
  - Search
  - Movie Details
